# Julia Oystrick - Portfolio Website

A modern, responsive portfolio website showcasing Julia Oystrick's professional profile, skills, education, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Easy Navigation**: Fixed navigation bar with smooth scrolling
- **Professional Photo Placeholder**: Hero section with initials display
- **Experience Timeline**: Showcase work experience and internships
- **Projects Portfolio**: Display business projects and case studies
- **Enhanced Skills**: Visual progress bars for skill levels
- **Certifications Section**: Highlight achievements and certifications
- **Contact Form**: Interactive contact form with email integration
- **Resume Download**: Downloadable resume/CV feature
- **Performance Optimized**: Lightweight and fast-loading

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

## Setup

This is a static website that can be hosted on GitHub Pages. Simply push the files to your repository and enable GitHub Pages in the repository settings.

## File Structure

```
JOystrick.github.io/
├── index.html          # Main HTML file
├── styles.css          # All styling
├── script.js           # JavaScript functionality
├── README.md           # This file
├── RESUME_NOTE.md      # Instructions for adding resume
└── resume.pdf          # Resume file (add when ready)
```

## Sections

1. **Hero**: Introduction with professional photo placeholder
2. **About**: Personal introduction and background
3. **Experience**: Work experience and internships timeline
4. **Projects**: Business projects and case studies showcase
5. **Skills**: Enhanced skills with visual progress indicators
6. **Education**: Academic background timeline
7. **Certifications**: Achievements and certifications
8. **Contact**: Contact form and direct contact methods

## Customization

To customize the portfolio:

1. **Colors**: Edit CSS variables in `styles.css` (lines 8-18)
2. **Content**: Update text in `index.html`
3. **Sections**: Add or remove sections as needed
4. **Styling**: Modify classes in `styles.css`
5. **Resume**: Add `resume.pdf` file to enable download feature
6. **Photo**: Replace hero image placeholder with actual photo
7. **Contact Form**: For production, integrate with Formspree, EmailJS, or Netlify Forms
